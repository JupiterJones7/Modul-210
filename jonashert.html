Begriffe im Bereich Cloud

  • Cloud: Ein Rechennetzwerk mit Server und Speicher Möglichkeiten
  • Public Cloud: Abbonierbarer Service, für jeden mit Zugangsrechten zugänglich
  • Private Cloud: Service, der von einem einzigem Organisator organisiert wird
  • Domain: Name, die auf IP-Adresse von einem Server zeigt Webserver: Server, der Files an Clients und Browser freigibt und überträgt
  • DNS: Übersetzt eingegebene Domain zu IP-Adresse
  • http: Protokoll, um Anfragen von Clients und Browser zu verarbeiten
  • FTP: File Transfare Protokoll, Dokumente unverschlüsselt auf einen Webserver zu laden
  • SFTP: SSH File Transfare Protokoll, überträgt Files per SSHS
